# Epyon

Epyon is a swiss army knife tool for pentesting DevOps ecosystems.

Available modules:

1. Gitlab
2. Github
3. Jenkins
4. Azure DevOps
5. Sonatype Nexus
6. Docker Registry
7. Sonarqube
8. Gitea
